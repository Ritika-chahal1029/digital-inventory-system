## Digital-inventory-system
A responsive web-based inventory management system built using HTML, CSS, and JavaScript. It enables users to manage products, track quantity, and store data persistently using browser local storage.

## Description
A simple web-based application to manage product inventory with add, view, and delete functionality.

## Features
- Add products with quantity
- Store data using localStorage
- Display timestamp (date & time)
- Delete products from list

## Tech Stack
- HTML
- CSS
- JavaScript

## How it works
Users can add product details which are stored in browser memory (localStorage) and displayed dynamically on the page.
