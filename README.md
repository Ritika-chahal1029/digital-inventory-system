# Digital Inventory Management System
## Digital inventory System


## Overview
I built a simple web-based Digital Inventory Management System to manage product data efficiently. This project allows users to add, view, and delete products with quantity tracking and stores data in the browser using local storage.

## Features
- Add new products with quantity
- Display product list dynamically
- Delete products from inventory
- Stores data using browser localStorage (persistent data)
- Shows date and time when product is added

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
Users can enter product name and quantity, which gets stored in the browser memory. The system dynamically updates the list and allows deletion of items. Data remains saved even after page refresh due to localStorage.

## Project Purpose
This project was created to understand basic web development concepts like DOM manipulation, event handling, and local storage in JavaScript.
